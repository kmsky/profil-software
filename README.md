The program creates database based on `.json` file generated by [randomuser.me](https://randomuser.me/). After that, allows to analyze some data from database.

It was created for the purpose of recruitment for the [Profil Software](https://profil-software.com/) company.

#### Requirements
* Python 3.7+
* Peewee (ORM)
* Requests library

#### Installation
Download and install requirements. Next run `database.py`  to create the database.

```sh
$ git clone https://github.com/k_msky/profil-software.git
$ cd profil-software
$ pip install -r requirements.txt
$ python database.py
```

#### Usage

Program is run by `main.py`, with arguments and parameters.
Supported arguments:
```
-percent-of-gender		# No parameter
-average-age			# Parameter: male/female/overall
-most-common-city		# Parameter: INT
-most-common-password		# Parameter: INT
-most-secure-password		# No parameter
-born-between			# Parameters: YYYY-MM-DD YYYY-MM-DD

# Example
$ python main.py -born-between 1965-10-02 1972-01-12
```
