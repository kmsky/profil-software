The program creates database based on data generated by [randomuser.me](https://randomuser.me/). After that, allows to analyze some data from database.

It was created for the purpose of recruitment for the [Profil Software](https://profil-software.com/) company.

### Requirements
* Python 3.7+
* Peewee (ORM)
* Requests library

### Installation
Download and install requirements. Next run `db_creator.py`.

```sh
$ git clone https://github.com/kmsky/profil-software.git
$ cd profil-software
$ pip install -r requirements.txt
$ python db_creator.py
```

### Usage

Program is run by `main.py`, with arguments and parameters.
Supported arguments:
```
-percent-of-gender		# No parameter
-average-age			# Parameter: male/female/overall
-most-common-city		# Parameter: INT
-most-common-password		# Parameter: INT
-most-secure-password		# No parameter
-born-between			# Parameters: YYYY-MM-DD YYYY-MM-DD

# Example
$ python main.py -born-between 1965-10-02 1972-01-12
```

### Important
Some of data is in Persian. To see them in CMD it is necessary to set a font that supports Persian characters, eg `Courier New`.
